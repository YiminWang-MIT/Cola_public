Programs and Commands

  • Cindy Online Analysis: Cola++
  • Display ECS Logfiles: Printlog++
  • The X11 Histogram viewer for HM-Histograms HMView
  • The Histogram Manager macro package HM
  • "hmdir filename" shows contents of histogram file.
  • "hmcat file1 file2 ... fileout" add up histogram files.
  • "hm2hbook filein fileout" convert to hbook format, use paw.
