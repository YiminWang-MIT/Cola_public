Compiling and Linking

Nearly all my programs use HMBook for pictures. You find the necessary include files at 
  /users/kpha1/merkel/hmbook
and the library in 
  /users/kpha1/merkel/decmips/lib          (Ultrix)
  /users/kpha1/merkel/alphaOSF1/lib        (Alpha's)
I suggest the following environment variables 
  export CC=gcc
  export CFLAGS=-Wall -ansi -pedantic -I/users/kpha1/merkel/hmbook
  export LDFLAGS=-L/users/kpha1/merkel/$ARCH/lib
  export LDLIBS=-lhmbook -lm
With these settings you should be able to create every program just by typing 
  make program
even if there is no Makefile available.
