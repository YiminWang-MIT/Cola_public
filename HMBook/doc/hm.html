Help for Histoman

Histoman is the histogram viewer macro packet of the HMBook software packet.

Command List:

  • Output file:
    • PLOTFILE "filename"
      open a file for grafics output
    • PIPE
      open a postscript interpreter (Ghostscript) as output device (Default)
    • CLOSE
       close output file or postscript interpreter
  • Histogram file and histogram handling:
    • READFILE "filename"
      open a histogram file
    • LIST, DIR, LS
      list the existing histograms
    • DELETE histogramnumber
      Delete a histogram
  • Histogram plotting
    • HISTOGRAM "histogramtitle"
      HISTOGRAM histogramnumber
      plot as a histogram
    • FUNCTION histogramnumber/title
      plot 1-D histogram as function
    • DATAERROR histogramnumber/title
      plot 1-D histogram as data points with error bars
    • GRAY histogramnumber/title
      plot 2-D histogram as grayscale plot
    • LEGO histogramnumber/title
      plot 2-D histogram as lego plot
    • BOX histogramnumber/title
      plot 2-D histogram as box plot
    • SCATTER histogramnumber/title
      plot 2-D histogram as lego plot
    • SURFACE histogramnumber/title
      plot 2-D histogram as surface
    • MINIMUMX
      set maximum for x axis for next plot
    • MINIMUMY
      etc.
    • MINIMUMZ
    • MAXIMUMX
    • MAXIMUMY
    • MAXIMUMZ
    • SAME
      draw next histogram into same frame
    • EXPLAIN "comment"
      a comment line for last histogram
  • Page layout
    • ZONE frames_in_x frames_in_y [first_frame]
      divide page into zones
    • TITLE "titleline"
      plot a title line for this page
    • SETOPTION option value
      set an option, e.g. fontsize, margins etc. See this table of options.
  • Misc.
    • HELP
      this help
    • WAIT
      wait
    • QUIT, EXIT, Q
      end program

Arithmetic operations between histograms (bin for bin):

Syntax: 

command:		@HISTOGRAMNUMBER = arithmetic_expression

arithmetic_expression : (arithmetic_expression)
 	                or arithmetic_expression + arithmetic_expression
 	                or arithmetic_expression - arithmetic_expression
 	                or arithmetic_expression / arithmetic_expression
 	                or arithmetic_expression * arithmetic_expression
	                histogram_id

histogram_id : 	        @HISTOGRAMNUMBER
	                or "Histogramtitle"
Use the command LIST to see the assigned histogram numbers. Only histogram numbers between 0 and 999 are allowed. Of course, arithmetic operations between histograms are only defined if the histograms are booked with the same bounderies and numbers of bins.

Example: 

@99  =  (@1 +@3)/(@3-@5*@123) 
@127 = "Cross Section" / "Phase Space"
